# EO Explorer v1.1

AI-Native Earth Observation Learning and Decision-Support Platform.

Explore satellite imagery from any location on Earth using Sentinel-2, Landsat 8/9,
and Sentinel-1 SAR. Assign any bands to R/G/B channels, compare all spectral views
at once, and get AI-powered explanations of what you are seeing.

## Features

- Any location — type any city, country, or region
- Three satellites: Sentinel-2 L2A, Landsat 8/9, Sentinel-1 SAR
- Full RGB band assignment — any band to any channel
- Named presets: True Color, Vegetation, Agriculture, Urban, Burn Scar, Water Bodies
- NDVI and NDWI computed server-side
- Compare All Views — all spectral combinations as thumbnails
- Scene timeline and coverage scoring
- 11-model AI fallback chain (Gemini first, Groq second)
- Works with no API keys in fallback mode
